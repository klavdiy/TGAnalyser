# TGAnalyser
 
> Аналитика Telegram-канала и разведка по публичным каналам. Без SaaS. Без подписок. Только ваши данные.
 
TGAnalyser — open-source инструмент на Python для глубокого анализа Telegram-каналов через официальный API.
Собирает полную историю постов с метриками вовлечённости, извлекает все исходящие ссылки,
картирует пересылки и выгружает всё в структурированный Excel-файл — без сторонних дашбордов
и без передачи прав администратора незнакомым ботам.
 
Построен на [Telethon](https://github.com/LonamiWebs/Telethon) и официальном Telegram API.
 
---
 
## Зачем это нужно
 
Встроенная аналитика Telegram — это количество подписчиков и охват за 7 дней.
Платные сервисы либо стоят денег, либо закрывают полезное за paywall, либо просят права администратора.
OSINT-инструменты заточены под разведку, а не под анализ собственного контента.
 
TGAnalyser работает локально, авторизуется от вашего аккаунта и не отправляет данные никуда.
 
---
 
## Возможности
 
- **Полная история постов** — просмотры, реакции с разбивкой по эмодзи, репосты, комментарии, engagement rate и составной Score
- **Извлечение исходящих ссылок** — все `t.me`-ссылки и `@упоминания` по всей истории канала, с дедупликацией и рейтингом по частоте
- **Карта пересылок** — откуда пересылали контент и как он сработал
- **Список участников** — там, где Telegram API это позволяет (группы и супергруппы)
- **Структурированный вывод** — CSV-файлы с временны́ми метками по типам данных + многолистовой Excel
- **Обработка FloodWait** — автоматический retry с задержкой, ручное вмешательство не нужно

## Структура вывода
 
```
TGSpyder_Output/
└── your_channel/
    ├── chats/
    │   └── messages_your_channel_20250507_143200.csv
    ├── crawled_links/
    │   └── crawled_links_your_channel_20250507_143200.csv
    ├── members/
    │   └── members_your_channel_20250507_143200.csv
    └── channel_stats_your_channel_20250507_143200.xlsx
```
 
Excel-файл содержит следующие листы:
 
| Лист | Содержимое |
|---|---|
| `Raw data` | Все посты со всеми метриками |
| `By Month` | Агрегаты по месяцам |
| `By Weekday` | Эффективность по дням недели |
| `By Hour` | Лучшее время публикации по средним просмотрам |
| `Top Posts` | Топ-20 постов по Score |
| `All Links` | Все исходящие TG-ссылки с привязкой к посту |
| `Top Links` | Ссылки, ранжированные по числу упоминаний |
| `Forwards` | Посты, пересланные из других каналов |
| `Members` | Список участников (где доступно) |

## Установка и запуск
 
**1. Получаем API-ключи**
 
Заходим на [my.telegram.org](https://my.telegram.org) → *API development tools* → создаём новое приложение с типом **Desktop**.
Получаем `api_id` (число) и `api_hash` (строка).
 
**2. Клонируем репозиторий**
 
```bash
git clone https://github.com/YOUR_USERNAME/tganalyser.git
cd tganalyser
```
 
**3. Создаём и активируем виртуальное окружение**
 
```bash
python3 -m venv venv
source venv/bin/activate        # macOS / Linux
venv\Scripts\activate           # Windows
```
 
**4. Устанавливаем зависимости**
 
```bash
pip install telethon pandas openpyxl tqdm
```
 
`telethon` — работа с Telegram API, `pandas` — обработка данных, `openpyxl` — запись в Excel, `tqdm` — прогресс-бар в терминале.
 
**5. Настраиваем скрипт**
 
Открываем `parser.py` и вписываем свои данные:
 
```python
API_ID   = 12345678          # ваш api_id
API_HASH = 'ваш_api_hash'    # ваш api_hash
CHANNEL  = 'имя_канала'      # без @
```
 
**6. Запускаем**
 
```bash
python parser.py
```
 
При первом запуске Telegram запросит номер телефона и код подтверждения — стандартная авторизация,
как в любом клиенте. Сессия сохраняется локально и переиспользуется при следующих запусках.

## Формула Score
 
Каждый пост получает составной балл для ранжирования:
 
```
Score = просмотры × 0.5 + реакции × 10 + репосты × 20
```
 
Репосты имеют наибольший вес как самый сильный органический сигнал.
Коэффициенты можно изменить в `parser.py` под свои задачи.

## Ограничения
 
- **Список участников** доступен только для групп и супергрупп. Публичные broadcast-каналы возвращают пустой результат — это ограничение Telegram API, не баг.
- **Реакции и просмотры** могут быть недоступны для постов старше ~2 лет на некоторых аккаунтах.
- Инструмент работает в рамках стандартных rate limits Telegram API.

## Этика и право
 
Инструмент предназначен для анализа каналов и групп, которыми вы владеете или администрируете.
Использование для парсинга сторонних каналов без разрешения может нарушать правила Telegram
и применимое законодательство. Автор не несёт ответственности за неправомерное использование.
